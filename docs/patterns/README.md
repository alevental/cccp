# Patterns

Reusable recipes for building pipelines and agents with CCCP.

## Format

Each pattern covers one technique. Required sections:

- **Problem** — what situation this addresses
- **Solution** — the pattern, with code/YAML snippets from the codebase
- **When to use / When NOT to use**
- **Examples** — real examples from `examples/` or `pipelines/`
- **Cross-links** — related docs

Patterns explain "how". ADRs explain "why".
